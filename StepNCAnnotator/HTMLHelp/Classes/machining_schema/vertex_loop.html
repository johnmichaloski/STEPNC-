ENTITY vertex_loop

A vertex loop is a loop of zero genus consisting of a single vertex.

Schema

machining_schema

Attributes

Attribute Type Description
loop_vertex vertex The vertex which defines the entire loop.

Express Definition

ENTITY vertex_loop 
SUBTYPE  OF (loop );
 loop_vertex : vertex;
END_ENTITY;
