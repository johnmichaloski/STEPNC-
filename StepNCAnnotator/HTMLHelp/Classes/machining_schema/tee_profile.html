ENTITY tee_profile

A T profile is a type of open profile the cross-section of which has the shape of the letter \"T\". The attributes are

Schema

machining_schema

Attributes

Attribute Type Description
first_angle plane_angle_measure Angular measurement for creating a chamfer on the open end.
second_angle plane_angle_measure Angular measurement for creating a chamfer between the stem and the cross bar parts of a T profile.
cross_bar_width toleranced_length_measure Width of the T cross bar.
cross_bar_depth toleranced_length_measure Depth of the T cross bar.
radius toleranced_length_measure Arc size for blending the sides of a tee cross bar.
width toleranced_length_measure Width of the T stem.
first_offset toleranced_length_measure Distance from the edge of the T stem to create a chamfer on the open end.
second_offset toleranced_length_measure Distance from the edge of the T stem to create a chamfer a distance from the edge of a surface to the finish of a chamfer.
