ENTITY toolpath_list

The entity toolpath_list contains a list of toolpaths (trajectories) which will be executed by the CNC one after the other.

Schema

machining_schema

Attributes

Attribute Type Description
its_list LIST [ 1 : ? ] OF toolpath A list of toolpaths. These toolpaths shall be executed in the defined order.

Express Definition

ENTITY toolpath_list ;
 its_list :  LIST [ 1 : ? ] OF toolpath;
END_ENTITY;
