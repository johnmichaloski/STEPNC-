ENTITY workpiece_probing

Probing of a dimension with one axis movement. Probing tool movement starts at start_position in the direction \"direction\" towards the workpiece. When the probe tool touches the workpiece, the machine stops and the difference of expected_value and tool position is stored in the inherited attribute measured_offset. Like machining_workingstep, workpiece_probing has a well defined start point and cannot be the first element in a workplan.

Schema

machining_schema

Attributes

Attribute Type Description
start_position axis2_placement_3d The starting position for the probing operation.
its_workpiece workpiece The workpiece to be probed.
its_direction direction The direction of the probing movement.
expected_value toleranced_length_measure The approximate value for probing.
its_probe touch_probe The identification of the probe which has to be used.

Express Definition

ENTITY workpiece_probing 
SUBTYPE  OF (touch_probing );
 start_position : axis2_placement_3d;
 its_workpiece : workpiece;
 its_direction : direction;
 expected_value : toleranced_length_measure;
 its_probe : touch_probe;
END_ENTITY;
