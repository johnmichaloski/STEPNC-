ENTITY chamfer

A chamfer is one of the two defined transition features for 2½D-machining This chamfer is always an outer chamfer as only this can be generated in an separate manufacturing operation.

Schema

machining_schema

Attributes

Attribute Type Description
angle_to_plane plane_angle_measure Angle between the first feature and the chamfer face measured from the first feature.
first_offset_amount toleranced_length_measure The offset of the chamfer measure from the edge of the first feature.

Express Definition

ENTITY chamfer 
SUBTYPE  OF (transition_feature );
 angle_to_plane : plane_angle_measure;
 first_offset_amount : toleranced_length_measure;
END_ENTITY;
