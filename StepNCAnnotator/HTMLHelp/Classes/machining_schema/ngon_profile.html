ENTITY ngon_profile

An ngon_ profile is a type of closed_profile that is an enclosed area bounded by three or more connected straight line sides. The orientation is at the center of the profile with one side of the ngon parallel to the X-axis crossing the Y-axis at a negative value.

Schema

machining_schema

Attributes

Attribute Type Description
diameter toleranced_length_measure The size of the circle which surrounds the ngon or is surrounded by the ngon. It depends on the flag of circumscribed_or_across_flats.
number_of_sides INTEGER Number of sides needed for the ngon.
circumscribed_or_across_flats BOOLEAN If false, the ngon is surrounded by a circle with the specified diameter. If true, the ngon surrounds a circle with the specified diameter.

Express Definition

ENTITY ngon_profile 
SUBTYPE  OF (closed_profile );
 diameter : toleranced_length_measure;
 number_of_sides :  INTEGER;
 circumscribed_or_across_flats :  BOOLEAN;
END_ENTITY;
