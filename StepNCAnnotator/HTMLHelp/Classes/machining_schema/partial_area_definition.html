ENTITY partial_area_definition

A partial_area_definition includes the limitations of a surface for applying a thread. It places a limitation on how much and where to apply the thread on the cylindrical shape.

Schema

machining_schema

Attributes

Attribute Type Description
effective_length length_measure The length of the thread which is usable.
placement axis2_placement_3d The location and direction of the partial_area_definition.
maximum_length length_measure The dimension along a surface to apply a thread. It limits The dimensional distance limits the length along the surface axis for defining a thread.

Express Definition

ENTITY partial_area_definition ;
 effective_length : length_measure;
 placement : axis2_placement_3d;
 maximum_length : OPTIONAL length_measure;
END_ENTITY;
