ENTITY address

An address is the place where people and organizations are located.

Schema

machining_schema

Attributes

Attribute Type Description
internal_location label an organization-defined address for internal mail delivery.
street_number label the number of a building in a street.
street label the name of a street.
postal_box label the number of a postal box.
town label the name of a town.
region label the name of a region. EXAMPLE - The counties of Great Britain and the states of the United States of America are examples of regions.
postal_code label the code that is used by the country's postal service.
country label the name of a country.
facsimile_number label the number at which facsimiles may be received.
telephone_number label the name of a country.
electronic_mail_address label  
telex_number label
