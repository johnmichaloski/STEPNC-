ENTITY cutter_contact_trajectory

Cutter contact paths define a tool movement with respect to the cutter contact point.

Schema

machining_schema

Attributes

Attribute Type Description
basiccurve curve_with_surface_normal A 3D curve defining the cartesian co-ordinates of the cutter contact point in the workpiece co
its_toolaxis bounded_curve A 2D curve defining the toolaxis of the tool. The first co-ordinate represents the yaw angle, the second co-ordinate the tilt angle in degrees. For the definition of tilt and yaw angles, see Section 5.2.2.3 of ISO 14649-11.. If given, the parameterisation of the toolaxis curve must correspond to that of the basiccurve. Note that this attribute will override any information about the tool orientation given by the its_tool_direction attribute of the workingstep.
its_contact_type contact_type  

Express Definition

ENTITY cutter_contact_trajectory 
SUBTYPE  OF (trajectory );
 basiccurve : curve_with_surface_normal;
 its_toolaxis : OPTIONAL bounded_curve;
 its_contact_type : OPTIONAL contact_type;
END_ENTITY;
