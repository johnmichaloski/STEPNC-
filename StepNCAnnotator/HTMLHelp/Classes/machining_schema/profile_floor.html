ENTITY profile_floor

A profile floor describes the bottom condition of a shape profile which may be flat or any arbitrary shape.

Schema

machining_schema

Attributes

Attribute Type Description
floor_radius numeric_parameter The radius of curvature for an arc between the bottom and the sides.
start_or_end BOOLEAN If true, the profile_floor is positioned at the end of a shape_profile. If false, its is at the start of the shape_profile.

Express Definition

ENTITY profile_floor  ABSTRACT
SUPERTYPE  OF ( ONEOF (general_profile_floor ,planar_profile_floor ) );
 floor_radius : OPTIONAL numeric_parameter;
 start_or_end :  BOOLEAN;
END_ENTITY;
