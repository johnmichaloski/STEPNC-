ENTITY feedstop

Feedstop is a toolpath element with no feed motion. The spindle remains in the selected speed while the execution of the subsequent toolpath is delayed for the specified time.

Schema

machining_schema

Attributes

Attribute Type Description
dwell time_measure The time to stop the feed axis.

Express Definition

ENTITY feedstop 
SUBTYPE  OF (toolpath );
 dwell : time_measure;
END_ENTITY;
