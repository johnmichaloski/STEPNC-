ENTITY rectangular_pattern

The description of elements arranged in a rectangular pattern. This may be either a grid of elements with n rows and m columns and a total number of n x m elements or a single line of m elements (n=1).

Schema

machining_schema

Attributes

Attribute Type Description
spacing toleranced_length_measure The spacing of the columns of the pattern. If there is only one row, this is also the spacing between the elements.
its_direction direction Direction of the first row measured from the direction given in feature_placement. It points into the direction of the row, describing the order to manufacture the replicated feature.
number_of_rows INTEGER The number of rows. Default is one, i. e. the rectangular pattern is a line of features. If number_of_rows is larger than one, the attributes row_spacing and row_layout_direction need to be specified.
number_of_columns INTEGER The number of columns.
row_spacing toleranced_length_measure The spacing of the rows (optional, needed if number_of_rows is larger than one). row_layout_direction: Optional description of the direction of the first column measured from the direction given in feature_placement. It points into the direction of the next row
row_layout_direction direction  
relocated_base_feature SET [ 0 : ? ] OF rectangular_offset Optional description of relocated features.
missing_base_feature SET [ 0 : ? ] OF rectangular_omit Optional description of missing features. Note that the number of features can be calculated from the number_of_rows, number_of_columns, missing_base_feature, and relocated_base_feature.

Express Definition

ENTITY rectangular_pattern 
SUBTYPE  OF (replicate_feature );
 spacing : toleranced_length_measure;
 its_direction : direction;
 number_of_rows : OPTIONAL  INTEGER;
 number_of_columns :  INTEGER;
 row_spacing : OPTIONAL toleranced_length_measure;
 row_layout_direction : OPTIONAL direction;
 relocated_base_feature :  SET [ 0 : ? ] OF rectangular_offset;
 missing_base_feature :  SET [ 0 : ? ] OF rectangular_omit;
END_ENTITY;
