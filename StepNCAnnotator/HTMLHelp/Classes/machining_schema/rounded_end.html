ENTITY rounded_end

A rounded end is a partially circular shape passed along a linear path.

Schema

machining_schema

Attributes

Attribute Type Description
course_of_travel linear_path A straight line with magnitude and direction.
partial_circular_boundary partial_circular_profile The arc that when swept along a path defines the area on a workpiece for volume removel.

Express Definition

ENTITY rounded_end 
SUBTYPE  OF (machining_feature );
 course_of_travel : linear_path;
 partial_circular_boundary : partial_circular_profile;
END_ENTITY;
