ENTITY curve_with_normal_vector

A bounded curve which has a second curve defining the surface normal.

Schema

machining_schema

Attributes

Attribute Type Description
basiccurve bounded_curve The reference to the geometric curve in the geometry description.
surface_normal bounded_curve A 3D curve in space defining the surface normal. The three co-ordinates of the curve represent the three components of the surface normal vector in the co-ordinate system of the basiccurve. It has to be of the same type and has to have the same parameterisation than the basiccurve.

Express Definition

ENTITY curve_with_normal_vector ;
 basiccurve : bounded_curve;
 surface_normal : bounded_curve;
END_ENTITY;
