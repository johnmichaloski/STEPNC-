ENTITY display_message

This function is used to display a message on the operator's screen.

Schema

machining_schema

Attributes

Attribute Type Description
its_text text The message to be displayed on the operator's screen, until the next message is sent. To clear the screen, send an empty message.

Express Definition

ENTITY display_message 
SUBTYPE  OF (nc_function );
 its_text : text;
END_ENTITY;
