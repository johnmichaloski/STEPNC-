ENTITY material

This entity is for identifying the workpiece material.

Schema

machining_schema

Attributes

Attribute Type Description
standard_identifier label The standard used for identifying the material. This can be a national standard or one used internally in the company.
material_identifier label The name which identifies the material.
material_property SET [ 0 : ? ] OF property_parameter The parameter which describes the properties of material. Since the demand for material properties varies, a generic type „property parameter” is used.

Express Definition

ENTITY material ;
 standard_identifier : label;
 material_identifier : label;
 material_property :  SET [ 0 : ? ] OF property_parameter;
END_ENTITY;
