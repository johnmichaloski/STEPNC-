ENTITY if_statement

The if statement runs an executable if a given condition is fulfilled; it may include an alternative option.

Schema

machining_schema

Attributes

Attribute Type Description
condition boolean_expression Expression to be tested based on the rules given below.
true_branch executable Executable to be executed if the condition evaluates to be true.
false_branch executable Executable to be executed if the condition evaluates to be false.

Express Definition

ENTITY if_statement 
SUBTYPE  OF (program_structure );
 condition : boolean_expression;
 true_branch : executable;
 false_branch : OPTIONAL executable;
END_ENTITY;
