ENTITY cutting_tool

Entity to describe the basic information needed for cutting tool description, which includes all type tools for chipping technologies (e.g. lathe tool, milling cutter, reamer, drill, tap, boring tool). As Part 10 cannot be implemented without at least one specific technology dependent part this Entity only includes general data and therefore is an abstract supertype.

Schema

machining_schema

Attributes

Attribute Type Description
its_tool_body tool_body The information describing the kind of cutting tool and it’s corresponding characteristics.
its_cutting_edge LIST [ 1 : ? ] OF cutting_component The information describing the cutting edge(s) of the cutting tool.
overall_assembly_length length_measure The entire length of the assembled tooling measured along the tooling axis for tool collision check.

Express Definition

ENTITY cutting_tool  ABSTRACT
SUPERTYPE 
SUBTYPE  OF (machining_tool );
 its_tool_body : tool_body;
 its_cutting_edge :  LIST [ 1 : ? ] OF cutting_component;
 overall_assembly_length : OPTIONAL length_measure;
END_ENTITY;
