ENTITY composite_curve

A composite curve is a collection of curves joined end­to­end. The individual segments of the curve are themselves defined as composite curve segments. The parametrisation of the composite curve is an accumulation of the parametric ranges of the referenced bounded curves.

Schema

machining_schema

Attributes

Attribute Type Description
segments LIST [ 1 : ? ] OF composite_curve_segment The component bounded curves, their transitions and senses. The transition attribute for the last segment defines the transition between the end of the last segment and the start of the first; this transition attribute may take the value discontinuous, which indicates an open curve.
self_intersect LOGICAL Indication of whether the curve intersects itself or not; this is for information only.

Express Definition

ENTITY composite_curve 
SUBTYPE  OF (bounded_curve );
 segments :  LIST [ 1 : ? ] OF composite_curve_segment;
 self_intersect :  LOGICAL;
END_ENTITY;
