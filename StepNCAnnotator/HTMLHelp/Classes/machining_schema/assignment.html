ENTITY assignment

Assigns a value to a nc_variable.

Schema

machining_schema

Attributes

Attribute Type Description
its_lvalue nc_variable The nc_variable which will be assigned a value.
its_rvalue rvalue The value to be assigned to its_lvalue. This is either a numeric constant or a reference to another nc_variable.

Express Definition

ENTITY assignment 
SUBTYPE  OF (program_structure );
 its_lvalue : nc_variable;
 its_rvalue : rvalue;
END_ENTITY;
