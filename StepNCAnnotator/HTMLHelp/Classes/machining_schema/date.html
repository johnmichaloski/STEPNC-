ENTITY date

A date is the identification of a day or week in a year.

Schema

machining_schema

Attributes

Attribute Type Description
year_component year_number the year in which the date occurs.

Express Definition

ENTITY date 
SUPERTYPE  OF ( ONEOF (calendar_date ,ordinal_date ,week_of_year_and_day_date ) );
 year_component : year_number;
END_ENTITY;
