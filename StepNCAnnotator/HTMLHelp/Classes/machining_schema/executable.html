ENTITY executable

Executable is the base entity of all executable objects. They initiate actions on a machine and shall be arranged in a defined order. There are three types of executable objects: workplans, nc_function, and workingstep.

Schema

machining_schema

Attributes

Attribute Type Description
its_id identifier The executable’s identifier. It shall be unique within the part program.

Express Definition

ENTITY executable  ABSTRACT
SUPERTYPE  OF ( ONEOF (workingstep ,nc_function ,program_structure ) );
 its_id : identifier;
END_ENTITY;
