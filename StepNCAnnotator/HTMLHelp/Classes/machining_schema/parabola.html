ENTITY parabola

A parabola is a conic section defined by its focal length, position (apex), and orientation.

Schema

machining_schema

Attributes

Attribute Type Description
focal_dist length_measure The distance of the focal point from the apex point.

Express Definition

ENTITY parabola 
SUBTYPE  OF (conic );
 focal_dist : length_measure;
END_ENTITY;
