ENTITY rectangular_closed_shape_profile

A rectangular_closed_shape_profile is a type of shape_profile that is an enclosed volume with opposite sides that are equal in length.

Schema

machining_schema

Attributes

Attribute Type Description
closed_boundary rectangular_closed_profile Specification of the outline or shape that is an enclosed area that has a completely closed profile.
