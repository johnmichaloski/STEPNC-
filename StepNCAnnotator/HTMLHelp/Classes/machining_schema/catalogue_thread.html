ENTITY catalogue_thread

A catalogue_thread is a type of thread that is defined by a document containing the information to create threads on a workpiece.

Schema

machining_schema

Attributes

Attribute Type Description
documentation specification Specification of the document that defines information pertaining to a thread.

Express Definition

ENTITY catalogue_thread 
SUBTYPE  OF (thread );
 documentation : specification;
END_ENTITY;
