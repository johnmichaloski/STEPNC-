ENTITY circular_path

The cicular_path is a type of path that is a direction of travel along an arc of constant radius around the Z-axis of the feature.

Schema

machining_schema

Attributes

Attribute Type Description
radius toleranced_length_measure The constant distance from an axis for the circular_path

Express Definition

ENTITY circular_path  ABSTRACT
SUPERTYPE  OF ( ONEOF (complete_circular_path ,partial_circular_path ) )
SUBTYPE  OF (travel_path );
 radius : toleranced_length_measure;
END_ENTITY;
