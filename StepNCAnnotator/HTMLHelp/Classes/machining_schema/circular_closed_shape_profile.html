ENTITY circular_closed_shape_profile

A circular_closed_shape_profile is a type of shape_profile that defines a completely enclosed volume. This may have a thread.

Schema

machining_schema

Attributes

Attribute Type Description
closed_boundary circular_closed_profile Specification of the outline of the shape_profile feature. It defines an area that shall be enclosed and circular.

Express Definition

ENTITY circular_closed_shape_profile 
SUBTYPE  OF (shape_profile );
 closed_boundary : circular_closed_profile;
END_ENTITY;
