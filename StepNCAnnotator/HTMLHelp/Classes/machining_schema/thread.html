ENTITY thread

A thread is a ridge of a uniform section on the form of a helix on the external or internal surface of a cylinder. Each thread is either a catalogue_thread or a defined_thread.

Schema

machining_schema

Attributes

Attribute Type Description
partial_profile partial_area_definition Specification of limitations of a surface to be applied on the thread.
applied_shape SET [ 1 : ? ] OF machining_feature Physical shape of the workpiece that will define where the thread will be applied.
inner_or_outer_thread BOOLEAN A flag specifies whether or not a thread is applied as an internal thread or an external thread.
qualifier descriptive_parameter Additional text information that describes a thread.
fit_class descriptive_parameter The value for the type of fit specification. These types are distinguished from each other by the amount of the tolerance and allowance.
form descriptive_parameter The definition of the shape of the thread. Various forms of threads are used to hold parts together, to adjust parts with reference to each other, or to transmit power.
major_diameter length_measure The dimension of the largest diameter of the thread and is applied to both an internal and external thread.
number_of_threads numeric_parameter The thread pitch when used with metric unit of measure and the density of threads per inch when used with English unit of measure.
thread_hand descriptive_parameter A description of whether the thread is right or left handed. When viewed toward an end, a right hand winds in a clockwise direction and a left hand winds in a counterclockwise direction.

Express Definition

ENTITY thread  ABSTRACT
SUPERTYPE  OF ( ONEOF (catalogue_thread ,defined_thread ) )
SUBTYPE  OF (machining_feature );
 partial_profile : partial_area_definition;
 applied_shape :  SET [ 1 : ? ] OF machining_feature;
 inner_or_outer_thread :  BOOLEAN;
 qualifier : OPTIONAL descriptive_parameter;
 fit_class : descriptive_parameter;
 form : descriptive_parameter;
 major_diameter : length_measure;
 number_of_threads : numeric_parameter;
 thread_hand : descriptive_parameter;
END_ENTITY;
