ENTITY swept_surface

A swept surface is one that is constructed by sweeping a curve along another curve

Schema

machining_schema

Attributes

Attribute Type Description
swept_curve curve The curve to be swept in defining the surface. If the swept curve is a pcurve, it is the image of this curve in 3D geometric space which is swept, not the parameter space curve.

Express Definition

ENTITY swept_surface 
SUPERTYPE  OF ( ONEOF (surface_of_linear_extrusion ,surface_of_revolution ) )
SUBTYPE  OF (surface );
 swept_curve : curve;
END_ENTITY;
