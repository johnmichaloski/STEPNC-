ENTITY b_spline_curve

A B­spline curve is a piecewise parametric polynomial or rational curve described in terms of control points and basis functions. The B­spline curve has been selected as the most stable format to represent all types of polynomial or rational parametric curves. With appropriate attribute values it is capable of representing single span or spline curves of explicit polynomial, rational, B'ezier or B­spline type. The b spline curve has three special subtypes where the knots and knot multiplicities can be derived to provide simple default capabilities.

Schema

machining_schema

Attributes

Attribute Type Description
degree INTEGER The algebraic degree of the basis functions.
control_points_list LIST [ 2 : ? ] OF cartesian_point The list of control points for the curve.
curve_form b_spline_curve_form Used to identify particular types of curve; it is for information only.
closed_curve LOGICAL Indication of whether the curve is closed; it is for information only.
self_intersect LOGICAL Flag to indicate whether the curve self­intersects or not; it is for information only.
