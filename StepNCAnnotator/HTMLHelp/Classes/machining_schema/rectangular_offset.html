ENTITY rectangular_offset

Entity to describe the position of a single relocated element whitin a rectangular pattern of elements.

Schema

machining_schema

Attributes

Attribute Type Description
offset_direction direction Direction of the offset of the element.
offset_distance length_measure Amount of offset of the element.
row_index INTEGER Row of the offset element.
column_index INTEGER Column of the offset element.

Express Definition

ENTITY rectangular_offset ;
 offset_direction : direction;
 offset_distance : length_measure;
 row_index :  INTEGER;
 column_index :  INTEGER;
END_ENTITY;
