ENTITY composite_curve_segment

A composite curve segment is a bounded curve together with transition information which is used to construct a composite curve.

Schema

machining_schema

Attributes

Attribute Type Description
transition transition_code The state of transition (i.e., geometric continuity from the last point of this segment to the first point of the next segment) in a composite curve.
same_sense BOOLEAN An indicator of whether or not the sense of the segment agrees with, or opposes, that of the parent curve. If same sense is false, the point with highest parameter value is taken as the first point of the segment.
parent_curve curve The bounded curve which defines the geometry of the segment.

Express Definition

 same_sense :  BOOLEAN;
