ENTITY linear_path

A linear_path is a type of path that is a direction of travel along a line.

Schema

machining_schema

Attributes

Attribute Type Description
distance toleranced_length_measure The length of the path.
its_direction direction A vector which indicates the direction of the path starting from the path placement.

Express Definition

ENTITY linear_path 
SUBTYPE  OF (travel_path );
 distance : toleranced_length_measure;
 its_direction : direction;
END_ENTITY;
