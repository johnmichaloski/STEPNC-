ENTITY ordinal_date

An ordinal_date is a date which is identified by a day of a year.

Schema

machining_schema

Attributes

Attribute Type Description
day_component day_in_year_number the day element of the date.

Express Definition

ENTITY ordinal_date 
SUBTYPE  OF (date );
 day_component : day_in_year_number;
END_ENTITY;
