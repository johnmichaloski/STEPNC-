ENTITY selective

The selective entity contains a set of executables from which only “one” will be executed. The other executables are to be omitted.

Schema

machining_schema

Attributes

Attribute Type Description
its_elements SET [ 2 : ? ] OF executable A set of executable objects of which exactly one has to be selected and executed by the controller of the machine tool.

Express Definition

ENTITY selective 
SUBTYPE  OF (program_structure );
 its_elements :  SET [ 2 : ? ] OF executable;
END_ENTITY;
