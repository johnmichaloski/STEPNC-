ENTITY face_bound

A face bound is a loop which is intended to be used for bounding a face.

Schema

machining_schema

Attributes

Attribute Type Description
bound loop The loop which will be used as a face boundary.
orientation BOOLEAN This indicates whether (TRUE), or not (FALSE) the loop has the same sense when used to bound the face as when first defined. If orientation is FALSE, the senses of all its component oriented edges are implicitly reversed when used in

Express Definition

ENTITY face_bound 
SUPERTYPE  OF (face_outer_bound )
SUBTYPE  OF (topological_representation_item );
 bound : loop;
 orientation :  BOOLEAN;
END_ENTITY;
