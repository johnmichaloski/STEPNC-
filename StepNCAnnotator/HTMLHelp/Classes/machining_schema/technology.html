ENTITY technology

This entity is the supertype for the technologies defined in the following parts of ISO 14649.

Schema

machining_schema

Attributes

Attribute Type Description
feedrate speed_measure Feed of the tool expressed as a linear speed. The feed rate specified applies to the motion of the tool center point.
feedrate_reference tool_reference_point Specifies whether the feed rate given above is to be calculated with regard to the tool center point or the cutter contact point.

Express Definition

ENTITY technology  ABSTRACT
SUPERTYPE ;
 feedrate : OPTIONAL speed_measure;
 feedrate_reference : tool_reference_point;
END_ENTITY;
