ENTITY local_time

time on a 24 hour clock. NOTE - This construct is used to represent a moment of time whereas time measures (see clause 20.4.1) represent amounts of time. EXAMPLE - 1500 hours is an instant in time whereas 15 hours is an amount of time.

Schema

machining_schema

Attributes

Attribute Type Description
hour_component hour_in_day the number of hours.
minute_component minute_in_hour the number of minutes.
second_component second_in_minute the number of seconds.
zone coordinated_universal_time_offset the relationship of the time to coordinated universal time.

Express Definition

ENTITY local_time ;
 hour_component : hour_in_day;
 minute_component : OPTIONAL minute_in_hour;
 second_component : OPTIONAL second_in_minute;
 zone : coordinated_universal_time_offset;
END_ENTITY;
