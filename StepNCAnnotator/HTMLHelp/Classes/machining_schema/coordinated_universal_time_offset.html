ENTITY coordinated_universal_time_offset

A coordinated_universal_time_offset is used to relate a time to coordinated universal time by an offset (specified in hours and minutes) and a direction.

Schema

machining_schema

Attributes

Attribute Type Description
hour_offset INTEGER the number of hours by which a time is offset from coordinated universal time.
minute_offset INTEGER the number of minutes by which a time is offset from coordinated universal time.
sense ahead_or_behind the direction of the offset.

Express Definition

ENTITY coordinated_universal_time_offset ;
 hour_offset :  INTEGER;
 minute_offset : OPTIONAL  INTEGER;
 sense : ahead_or_behind;
END_ENTITY;
