ENTITY vertex_point

A vertex point is a vertex which has its geometry defined as a point.

Schema

machining_schema

Attributes

Attribute Type Description
vertex_geometry point The geometric point which defines the position in geometric space of the vertex.

Express Definition

ENTITY vertex_point 
SUBTYPE  OF (vertex,geometric_representation_item );
 vertex_geometry : point;
END_ENTITY;
