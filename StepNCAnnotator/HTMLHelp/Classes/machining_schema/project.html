ENTITY project

Each part programme, i.e. data model, based on ISO14649 must include exactly one top-level entity, called project. The project indicates the workplan to be executed upon interpretation of this model (as several workplans might be included), and it may also provide the workpiece(s) upon which actions are to be performed.

Schema

machining_schema

Attributes

Attribute Type Description
its_id identifier The project’s identifier. It shall be unique within the part programme.
main_workplan workplan The top-level workplan in this model.
its_workpieces SET [ 0 : ? ] OF workpiece The workpieces upon which actions are to be performed.
its_owner person_and_address Optional information on the owner of the project.
its_release date_and_time Optional date and time reference of the project.
its_status approval Optional attribute to indicate the current status of the project.

Express Definition

ENTITY project ;
 its_id : identifier;
 main_workplan : workplan;
 its_workpieces :  SET [ 0 : ? ] OF workpiece;
 its_owner : OPTIONAL person_and_address;
 its_release : OPTIONAL date_and_time;
 its_status : OPTIONAL approval;
END_ENTITY;
