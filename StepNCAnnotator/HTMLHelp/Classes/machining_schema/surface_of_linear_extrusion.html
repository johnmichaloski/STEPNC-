ENTITY surface_of_linear_extrusion

This surface is a simple swept surface or a generalised cylinder obtained by sweeping a curve in a given direction.

Schema

machining_schema

Attributes

Attribute Type Description
extrusion_axis vector The direction of extrusion, the magnitude of this vector determines the parameterization.

Express Definition

ENTITY surface_of_linear_extrusion 
SUBTYPE  OF (swept_surface );
 extrusion_axis : vector;
END_ENTITY;
