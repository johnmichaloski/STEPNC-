ENTITY date_and_time

A date_and_time is a moment of time on a particular day.

Schema

machining_schema

Attributes

Attribute Type Description
date_component date the date element of the date time combination.
time_component local_time the time element of the date time combination.

Express Definition

ENTITY date_and_time ;
 date_component : date;
 time_component : local_time;
END_ENTITY;
