ENTITY edge_round

An edge round is the other of the two defined transition features. In 2½D-machining, it is generated using a contoured tool which has to be selected according to the given fillet radius. The edge_round is always an outer fillet, since only this type of fillet can be manufactured in an extra manufacturing operation.

Schema

machining_schema

Attributes

Attribute Type Description
radius toleranced_length_measure Radius of the edge round.
first_offset_amount toleranced_length_measure The edge_round may or may not be tangent to the two features, this attribute specifies the location of the edge round
second_offset_amount toleranced_length_measure A non-tangent edge_round may or may not be symmetrical to the two features. If it is not symmetrical, this feature specifies the location of the edge round

Express Definition

ENTITY edge_round 
SUBTYPE  OF (transition_feature );
 radius : toleranced_length_measure;
 first_offset_amount : OPTIONAL toleranced_length_measure;
 second_offset_amount : OPTIONAL toleranced_length_measure;
END_ENTITY;
