ENTITY circle

A circle is a conic section defined by a radius and the location and orientation of the circle.

Schema

machining_schema

Attributes

Attribute Type Description
radius positive_length_measure radius of the circle, which shall be greater than zero.

Express Definition

ENTITY circle 
SUBTYPE  OF (conic );
 radius : positive_length_measure;
END_ENTITY;
