ENTITY general_shape_profile

A general_shape_profile is a type of shape_profile that is a volume of arbitrary shape which defines a portion of the workpiece.

Schema

machining_schema

Attributes

Attribute Type Description
profile_boundary profile Specification of the outline of the shape_profile feature. It defines an area that may or may not be entirely enclosed.

Express Definition

ENTITY general_shape_profile 
SUBTYPE  OF (shape_profile );
 profile_boundary : profile;
END_ENTITY;
