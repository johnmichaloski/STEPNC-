ENTITY polyline

A polyline is a bounded curve of n \Gamma 1 linear segments, defined by a list of n points,

Schema

machining_schema

Attributes

Attribute Type Description
points LIST [ 2 : ? ] OF cartesian_point The cartesian points defining the polyline.

Express Definition

ENTITY polyline 
SUBTYPE  OF (bounded_curve );
 points :  LIST [ 2 : ? ] OF cartesian_point;
END_ENTITY;
