ENTITY nc_constant

The nc_constant is used to assign a constant value to an attribute. Its value is determined by the NC-Program and cannot be modified within the process.

Schema

machining_schema

Attributes

Attribute Type Description
its_name label Name of the constant. It shall be unique within the program.
its_value number The value of the nc_constant.

Express Definition

ENTITY nc_constant ;
 its_name : label;
 its_value : OPTIONAL number;
END_ENTITY;
