ENTITY trajectory

Supertype for all explicit trajectories.

Schema

machining_schema

Attributes

Attribute Type Description
its_direction BOOLEAN If true or omitted, the tool has to be moved from the beginning to the end of the referenced geometric curve. Otherwise, it will be moved from the end point to the start point of the referenced geometric curve.

Express Definition

ENTITY trajectory  ABSTRACT
SUPERTYPE  OF ( ONEOF (cutter_location_trajectory ,cutter_contact_trajectory ,axis_trajectory ) )
SUBTYPE  OF (toolpath );
 its_direction : OPTIONAL  BOOLEAN;
END_ENTITY;
