ENTITY non_sequential

The non_sequential entity allows to define a set of executables which all shall be executed but which’s order is not prescribed. In contrast to the workplan entity the non_sequential does not define a sequence.

Schema

machining_schema

Attributes

Attribute Type Description
its_elements SET [ 2 : ? ] OF executable An set of executable objects. The non_sequential cannot contain itself as element. It can only contain itself indirectly within a control structure which allows to skip itself in order to prevent an infinite loop.

Express Definition

ENTITY non_sequential 
SUBTYPE  OF (program_structure );
 its_elements :  SET [ 2 : ? ] OF executable;
END_ENTITY;
