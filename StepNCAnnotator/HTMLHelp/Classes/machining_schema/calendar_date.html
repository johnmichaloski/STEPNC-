ENTITY calendar_date

A calendar_date is a date which is identified by a day in a month of a year.

Schema

machining_schema

Attributes

Attribute Type Description
day_component day_in_month_number the day element of the date.
month_component month_in_year_number the month element of the date.

Express Definition

ENTITY calendar_date 
SUBTYPE  OF (date );
 day_component : day_in_month_number;
 month_component : month_in_year_number;
END_ENTITY;
