ENTITY general_profile_floor

A general_profile_floor is a type of profile_floor that specifies an enclosed area bounded by an arbitrary shape.

Schema

machining_schema

Attributes

Attribute Type Description
floor face Specification of the face at the bottom of a shape_profile, adjacent to all the shape_profile walls.

Express Definition

 floor : face;
END_ENTITY;
