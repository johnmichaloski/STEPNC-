ENTITY offset_vector

This entity is used to store the offset resulting from a workpiece probing operation. All variables must have initial values, most likely zeros.

Schema

machining_schema

Attributes

Attribute Type Description
translate LIST [ 3 : 3 ] OF nc_variable Variables to store the translation required in each co-ordinate direction (x-, y-, z- axis) in order to move from the theoretical position to the actual position, measured in millimetres.
rotate LIST [ 3 : 3 ] OF nc_variable Variables to store the rotation around the x-, y-, and z-axis, respectively, in order to move from the theoretical position to the actual position, measured in degrees. If omitted, only the translational offset will be considered.

Express Definition

ENTITY offset_vector ;
 translate :  LIST [ 3 : 3 ] OF nc_variable;
 rotate : OPTIONAL  LIST [ 3 : 3 ] OF nc_variable;
END_ENTITY;
