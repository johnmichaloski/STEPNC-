ENTITY specification

A specification is a document that defines information pertaining to properties or processes for a workpiece or an aspect of a workpiece.

Schema

machining_schema

Attributes

Attribute Type Description
constraint SET [ 0 : ? ] OF specification_usage_constraint A set of the restrictions on the specification.
specification_id text A unigue identifier of the document.
specification_description text A description of the content of the specification and any notes in human interpretable prose.
specification_class text A section within a specification that is divided into classes.

Express Definition

ENTITY specification ;
 constraint :  SET [ 0 : ? ] OF specification_usage_constraint;
 specification_id : text;
 specification_description : OPTIONAL text;
 specification_class : OPTIONAL text;
END_ENTITY;
