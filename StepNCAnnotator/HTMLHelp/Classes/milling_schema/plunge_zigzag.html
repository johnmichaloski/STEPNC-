ENTITY plunge_zigzag

Plunge movement using a zigzag motion. This is similar to the ramp-type movement, except the cutter changes direction if it touches a feature boundary or if the path length would exceed the specified width of the zigzag pattern.

Schema

milling_schema

Attributes

Attribute Type Description
angle plane_angle_measure The angle of the movement versus the surface in the end point of the approach.
width length_measure The with of the zigzag path perpendicular to the direction of the descent.

Express Definition

ENTITY plunge_zigzag 
SUBTYPE  OF (plunge_strategy );
 angle : plane_angle_measure;
 width : length_measure;
END_ENTITY;
