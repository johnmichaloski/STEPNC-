ENTITY milling_machine_functions

The entity describes the state of various functions of the machine, like coolant, chip removal, etc. to be applied during the time span of an operation. It is a subtype of entity machine_functions defined in ISO 14649-10.

Schema

milling_schema

Attributes

Attribute Type Description
coolant BOOLEAN If true, the coolant is activated.
coolant_pressure pressure_measure Optional specification of the pressure of the coolant system. Only valid if coolant is true.
mist BOOLEAN If true, activate mist coolant. Default is false. Only valid if coolant is true.
through_spindle_coolant BOOLEAN If true, activate coolant through the spindle. Default is false.
through_pressure pressure_measure Pressure of coolant through the spindle. Only valid if through_spindle_coolant is true.
axis_clamping LIST [ 0 : ? ] OF identifier Describes which axes are to be clamped, e.g. X,Y,A. Note that this information is machine dependent and should be avoided.
chip_removal BOOLEAN If true, activate chip removal.
oriented_spindle_stop direction If specified, the spindle will stop in the given direction relative to the machine zero position of C-axis in case a spindle stop occurs during or at the end of the workingstep.
its_process_model process_model_list Optional information for process control.
other_functions SET [ 0 : ? ] OF property_parameter Optional list of other functions of generic type.
