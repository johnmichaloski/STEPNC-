ENTITY milling_technology

This entity defines the technological parameters of the milling operation. It is a subtype of entity technology defined in ISO 14649-10. Of the four alternatives for specifying speeds, exactly two must be given as indicated by the WHERE rules. If the attribute adaptive_control s invoked, some or all of these values may be ignored.

Schema

milling_schema

Attributes

Attribute Type Description
cutspeed speed_measure Cutting speed of the tool, the speed of spindle converted into a linear speed.
spindle rot_speed_measure Rotational speed of the tool. As defined for rot_speed_measure, positive values indicate tool rotation in mathematical positive direction of the c axis, i. e. counterclockwise motion if looking from the tool holder to the workpiece. Note that usual cutting tools require clockwise motion so the value of this attribute will typically be negative.
feedrate_per_tooth length_measure Feed of the tool expressed as a distance.
synchronize_spindle_with_feed BOOLEAN If true, cutting speed and feed of the tool is synchronised. Therefore, the pitch of tap can be kept constant at the bottom of a hole when cutting speed is being decelerated and accelerated.
inhibit_feedrate_override BOOLEAN If true, the feedrate override through the operating panel or by adaptive control systems is not allowed.
inhibit_spindle_override BOOLEAN If true, the spindle speed override through the operating panel or by adaptive control systems is not allowed.
its_adaptive_control adaptive_control Any kind of vendor specific adaptive control strategy.

Express Definition
