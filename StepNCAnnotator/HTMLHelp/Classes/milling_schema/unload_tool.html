ENTITY unload_tool

This function is used to unload a tool.

Schema

milling_schema

Attributes

Attribute Type Description
its_tool machining_tool The tool which has to be exchanged. In case of an operation where more than one tool is in use at the same time this attribute has to be set.

Express Definition

ENTITY unload_tool 
SUBTYPE  OF (nc_function  );
 its_tool : OPTIONAL machining_tool;
END_ENTITY;
