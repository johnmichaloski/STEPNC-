ENTITY uv_strategy

Milling follows the parameter lines in the local (u,v) coordinate system.

Schema

milling_schema

Attributes

Attribute Type Description
forward_direction direction The direction used in the first cut.
sideward_direction direction The direction in which the second cut is offset from the first.

Express Definition

ENTITY uv_strategy

END_ENTITY;
