ENTITY side_milling

Entity to describe a side milling process during which material is removed along the flank of the tool.

Schema

milling_schema

Attributes

Attribute Type Description
axial_cutting_depth length_measure The cutting depth in the direction of the tool axis. See plane milling.
radial_cutting_depth length_measure The cutting depth perpendicular to the tool axis. This can be used to limit the chip thickness. If radial_cutting_depth is smaller than the radial material removal required by the feature, this will cause the execution of the operation in several layers.
allowance_side length_measure The allowance is a layer of material which will be left on side of the surface defined by the associated manufacturing feature.

Express Definition

ENTITY side_milling  ABSTRACT
SUPERTYPE  OF ( ONEOF (side_rough_milling ,side_finish_milling ) )
SUBTYPE  OF (two5d_milling_operation );
 axial_cutting_depth : OPTIONAL length_measure;
 radial_cutting_depth : OPTIONAL length_measure;
 allowance_side : OPTIONAL length_measure;
END_ENTITY;
