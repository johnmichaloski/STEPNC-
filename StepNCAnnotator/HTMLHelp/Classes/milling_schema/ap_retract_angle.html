ENTITY ap_retract_angle

The movement is heading towards the start or from the end point in an angle to the surface. For plane milling, this may typically be an angle of 0 degrees in order to move straight from outside the workpiece into the material.

Schema

milling_schema

Attributes

Attribute Type Description
angle plane_angle_measure Approach or lift angle versus the surface in the end point of the approach or the start point of the lift, respectively.
travel_length length_measure The length of the angular approach. After travel_length has been reached, the tool will proceed to the retract plane using the shortest connection and vice versa.

Express Definition

ENTITY ap_retract_angle 
SUBTYPE  OF (air_strategy );
 angle : plane_angle_measure;
 travel_length : length_measure;
END_ENTITY;
