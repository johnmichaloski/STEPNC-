ENTITY process_model

Special machine-specific functions to make the machining process more secure and accurate. (e.g. chatter avoidance, thermal compensation, ...)

Schema

milling_schema

Attributes

Attribute Type Description
ini_data_file label A filename including path of the file containing the initialisation data of the process model.
its_type label The type of process model (e.g. chatter avoidance, thermal compensation, ...)

Express Definition

ENTITY process_model ;
 ini_data_file : label;
 its_type : label;
END_ENTITY;
