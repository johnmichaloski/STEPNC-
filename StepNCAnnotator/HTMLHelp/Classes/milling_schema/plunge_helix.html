ENTITY plunge_helix

Plunge movement forming a helix. The path is defined by specifying the radius and grade of the helix. A circular movement can be specified by setting grade to zero.

Schema

milling_schema

Attributes

Attribute Type Description
radius length_measure Radius of the helical movement.
angle plane_angle_measure The angle of the helical movement versus the surface in the end point of the approach.

Express Definition

ENTITY plunge_helix 
SUBTYPE  OF (plunge_strategy );
 radius : length_measure;
 angle : plane_angle_measure;
END_ENTITY;
