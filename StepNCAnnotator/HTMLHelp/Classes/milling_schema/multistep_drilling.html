ENTITY multistep_drilling

Workingstep for drilling of deep holes in multiple steps.

Schema

milling_schema

Attributes

Attribute Type Description
retract_distance length_measure If retract_distance is a positive value, the tool is retracted this value between steps to enable chip breaking. If it is zero or negative, the tool is retracted to the retract plane between steps to clear the hole of chips.
first_depth length_measure Depth of the first step.
depth_of_step length_measure Depth of each additional step (repeated until the depth of the hole is reached).
dwell_time_step time_measure Dwell time between steps.

Express Definition

ENTITY multistep_drilling 
SUBTYPE  OF (drilling_operation );
 retract_distance : length_measure;
 first_depth : length_measure;
 depth_of_step : length_measure;
 dwell_time_step : OPTIONAL time_measure;
END_ENTITY;
